⏱️ Stopwatch Web Application

A simple and interactive Stopwatch Web Application built using HTML, CSS, and JavaScript. This project allows users to accurately measure time, pause and resume the stopwatch, reset it, and record lap times through an intuitive and user-friendly interface.

🚀 Features

- ▶️ Start the stopwatch
- ⏸️ Pause the stopwatch
- 🔄 Reset the stopwatch
- 🏁 Record multiple lap times
- ⏱️ Display time in Hours : Minutes : Seconds : Milliseconds
- 📱 Responsive and clean user interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

📂 Project Structure

```
Stopwatch/
│── index.html
│── style.css
└── script.js
```

▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. Use the Start, Pause, Reset, and Lap buttons to operate the stopwatch.

🎯 Project Objective

The objective of this project is to develop an interactive stopwatch application that enables users to measure elapsed time accurately. The application provides essential stopwatch functionalities, including start, pause, reset, and lap recording, while offering a responsive and user-friendly interface.

📸 Output

The application displays:
- Live stopwatch timer
- Start, Pause, Reset, and Lap buttons
- Recorded lap times
